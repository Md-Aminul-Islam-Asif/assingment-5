Important FAQ

What is DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, and text). This allows programming languages like JavaScript to interact with, manipulate, and modify the document's content, structure, and style dynamically.

How do you select an element from the DOM?

Selecting an element from the DOM is commonly done using JavaScript. The most popular methods for DOM selection are based on element attributes like ID, class, tag name, and selectors

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to handle events that occur on its child elements. Instead of adding individual event listeners to each child element, the parent element captures the event as it "bubbles up" from the child. This technique leverages the event bubbling mechanism, where events propagate from the innermost element (the event target) up to the outermost (the root of the DOM).

How do you manipulate and element's attributes and style using the DOM?

To manipulate an element's attributes, use getAttribute(), setAttribute(), or removeAttribute(). For styles, modify the style property directly (e.g., element.style.color = "red") or use classList methods to add, remove, or toggle CSS classes. You can also handle data attributes via the dataset property (e.g., element.dataset.info).
